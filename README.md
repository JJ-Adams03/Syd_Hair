## Installation & Deployment

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/[username]/Syd_Hair.git
   cd Syd_Hair
   ```

2. Serve the files using a local web server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Or simply open index.html in your browser
   ```

3. Visit `http://localhost:8000` in your browser

### Production Deployment

The website is ready for deployment to any web hosting service:

- **GitHub Pages:** Automatically deployed from the main branch
- **Netlify:** Drag and drop the project folder
- **Vercel:** Connect your GitHub repository
- **Traditional Hosting:** Upload all files via FTP

### Required Images

To complete the website, add these images to the `/images/` directory:

- `syd-stylist.jpg` - Professional photo of Syd for the About section
- `salon-hero.jpg` - Hero background image (optional)
- `favicon.ico` and `apple-touch-icon.png` - Site icons
